CSS Variables

10 — Conventions

Usage Notes

  • Font stack is 'Segoe UI', Arial, sans-serif — no web font imports required.
  • All interactive elements use border-radius: 4px — cards, buttons, code blocks, preview boxes.
  • Transitions are uniformly 0.2s ease for color/opacity and 0.15s for transforms.
  • The hero banner is always dark (var(--black)) and appears directly below the header on white-background pages.
  • Section labels (.section-label) always appear above the section title in var(--cyan) small caps.
  • Every major section ends with a .divider component (cyan square + horizontal rule).
  • Card hover state uses a ::before pseudo-element for the animated cyan top bar — do not add a real top border.
  • Main content is constrained to max-width: 1100px and centered with margin: 0 auto.
  • The footer always uses background: var(--black) with border-top: 3px solid var(--cyan).
  • All uppercase labels use letter-spacing: 0.12em or higher — never use font-variant: small-caps.
  • Box shadows are subtle: 0 2px 8px rgba(0,0,0,0.07) for header, 0 6px 24px rgba(0,0,0,0.10) for hovered cards.
  • Always reference colors via CSS variables (var(--cyan)) — never hardcode hex values in component styles.
