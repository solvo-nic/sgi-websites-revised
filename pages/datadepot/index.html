DataDepot Overview

DataDepot provides the means to review the stock that SGI manages for you. The DataDepot web site provides two features to assist in finding and assessing information with:

  • Dashboard
  • Smart Search

The DataDepot web site provides you with access to information about:

  • Alerts
  • Items
  • Orders
  • Receipts
  • Utilization

Use DataDepot to assist you in making decisions about the inventory. DataDepot provides the information you need to reduce material and storage costs.
